# 과제 - 03. 연산자와 표현식

## 문제 1 — BMI 계산기
- 프로젝트 폴더: `Homework01/`
- 핵심 개념: `double` 연산, 비교 연산자, 문자열 보간 `:F2`

### 요구사항
- 키(cm)와 몸무게(kg)를 입력받는다
- BMI = `몸무게 / (키[m] × 키[m])` 으로 계산한다
- 소수점 둘째 자리까지 출력한다 (`{bmi:F2}`)

### 예상 출력
```
키(cm): 172
몸무게(kg): 70
BMI: 23.66
```

### 힌트
- cm 를 m 로 바꾸려면 `/ 100.0` (꼭 `100.0` 으로 — `double` 보장)
- `double.Parse(Console.ReadLine()!)` 로 실수 입력

## 문제 2 — 짝수/홀수 판별 (삼항 연산자)
- 프로젝트 폴더: `Homework02/`
- 핵심 개념: `%`, 삼항 연산자 `?:`

### 요구사항
- 정수 하나를 입력받는다
- 삼항 연산자로 짝수/홀수를 판별해 출력한다
- 0 을 입력하면 "0 은 짝수입니다" 라고 출력한다 (0 은 짝수로 처리)

### 예상 출력
```
정수: 7
7 은 홀수입니다.
```

### 힌트
- `n % 2 == 0` 이면 짝수
- `string result = ... ? "짝수" : "홀수";`
